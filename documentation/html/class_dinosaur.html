Detailed Description

This class constructs a dinosaur.

It is a base class for Carnivorous, Herbivorous, Aqueous and Flying.

The variables(data members of the class) which are declared in Dinosaur are:

Data members:

  • char* dinosaurName - this is the name of dinosaur.
  • Gender genderOfDinosaur - this is the gender of dinosaur. It's enum variable : Male or Female.
  • static const char* gender - this is static variable for gender. We use this static variable to index through the variable enum. Enum gives us the indexes, and the variable returns the gender.
  • Era era - era of dinosaurs. It's enum variable : Triassic , Cretaceous or Jurassic
  • static const char* _era - this is static variable for era. We use this static variable to index through the variable enum. Enum gives us the indexes, and the variable returns the era.
  • char* typeOfDinosaur - the type of dinosaur.
