Detailed Description

This class constructs a carnivorous dinosaur.

This is the class that inherits from Dinosaur class. It called derived class.

The variables(data members of the class, which ARE NOT in Dinosaur) are:

  • Food foodThatCarnivorousDinosaurEats - this is the food that dinosaur eats. It's enum variable : Grass,Meat or Fish.
  • static const char* food - this is static variable for food. We use this static variable to index through the variable enum. Enum gives us the indexes, and the variable returns the food that dinosaur eats.
  • Order order - the order of dinosaur. It's enum variable : Order::Carnivorous, Order::Herbivorous, Order::Aqueous or Order::Flying.
  • static const char* _order - this is static variable for order. We use this static variable to index through the variable enum. Enum gives us the indexes, and the variable returns the order of dinosaur.

Parameters

typeOfDinosaur-> this is the type that your dinosaur is going to be;
Attention
